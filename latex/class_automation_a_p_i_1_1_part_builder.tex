\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_automation_a_p_i_1_1_part_builder}{Part\+Builder}} is used by the Automation API to create spherical parts. 

This part builder creates spherical parts either by using a given radius, or by using a given circumference. Once the part is created you can change the part using Set\+Radius, Set\+Origin, or Set\+Circumference. To get info about the part you can use Get\+Radius, Get\+Origin, Get\+Circumference, Get\+Surface\+Area, or Get\+Volume.
