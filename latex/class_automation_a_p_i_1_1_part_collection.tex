\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_automation_a_p_i_1_1_part_collection}{Part\+Collection}} is used by the Automation API to create a part builder 

This creates a part builder which is capable of creating spherical parts. Expect this to be expanded to other part types in the future. To start the part builder you must provide a pointer to a part, or a nullptr.
